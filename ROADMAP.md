# Roadmap

## Phase 1: Foundation (Current)
- [x] Core scraping engine (static + JS-rendered)
- [x] Multi-service architecture (API, Worker, Web, Scraper)
- [x] BullMQ job queue with Redis
- [x] Railway deployment
- [x] Docker Compose for local dev
- [x] Basic CI/CD pipelines

## Phase 2: Quality & Community
- [ ] Add comprehensive test coverage (>80%)
- [ ] Publish OpenAPI/Swagger documentation
- [ ] Add GitHub badges and topics
- [ ] Publish demo site
- [ ] Write blog post / announcement
- [ ] Add issue and PR templates

- [ ] User authentication (OAuth + JWT)
- [ ] API key management per user
- [ ] Job scheduling (cron-based recurring scrapes)
- [ ] Webhook notifications on job completion
- [ ] Data export (CSV, JSON, Parquet)
- [ ] Rate limiting per user (not just IP)
